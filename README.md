### 🧰 项目启动
**1、⚙ 项目环境配置**
```shell
# 拉取项目
git clone https://github.com/DDongAI/fastdata.git
```
```shell
cd fastdata
```
```shell
# 配置环境变量
cp .env.example .env
# 配置模型
vi .env
```
```shell
# 修改其他信息
vi docker-compose.yml
vi Dockerfile
```
**2、⚙ 启动命令**
- 🛠 方式1：
```shell
cd fastdata
```
```shell
docker-compose up -d
```
